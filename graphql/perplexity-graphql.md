# Perplexity GraphQL API

Perplexity AI is a conversational AI search engine. Their API provides access to online and offline AI models for answering questions with real-time search grounding, citation attribution, and streaming responses.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Perplexity REST API
